# 💰 Peko – AI Expense Tracker

**Peko** is an intelligent expense tracking app built with **Flutter** and powered by **AI** to help users manage their money in a smarter and simpler way.

---

## 🚀 Overview

Peko goes beyond traditional expense tracking apps.

It uses **Artificial Intelligence** to analyze your income, expenses, and spending habits — then generates **personalized financial plans** to help you save more and spend wisely.

---

## ✨ Features

* 📊 Easy income & expense tracking
* 🤖 AI-powered financial planning
* 🧠 Smart analysis of spending behavior
* 🔐 Firebase Authentication (Email & Google Sign-In)
* ☁️ Cloud Firestore integration
* 📈 Financial statistics & insights
* 🏗️ Clean Architecture (scalable & maintainable)

---

## 🧱 Architecture

This project follows **Clean Architecture** principles:

```bash id="9y0b8y"
lib/
 ├── core/
 ├── features/
 │    ├── auth/
 │    ├── transaction/
 │    ├── budget/
 │    ├── stats/
 │    └── profile/
 ├── shared/
 └── routes/
```

### Layers:

* **Presentation** → UI + Bloc
* **Domain** → Business Logic
* **Data** → Firebase & APIs

---

## 🛠️ Tech Stack

* Flutter
* Dart
* Firebase (Authentication & Firestore)
* Bloc (State Management)
* AI Integration (Gemini / Custom API)

```bash id="4u9g9s"
cd peko
flutter pub get
flutter run
```

---

## 🔑 Firebase Setup

1. Create a Firebase project
2. Add your Android app
3. Download `google-services.json`
4. Place it inside:

```bash id="o5d5nn"
android/app/
```

5. Enable:

* Authentication (Email + Google)
* Firestore Database
